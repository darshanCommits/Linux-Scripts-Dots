# System Wide Keymap
- capslock 
  - hold : control
  - click : escape
- insert : S-insert

## Layers
- alt (left)
  - h/j/k/l : vim bind
  - a : select all / C-a
  - c/v : copy/pase
  - q : delete word forward
  - d : delete char forward

- control (Emacs readline mode)
  - h : backspace
  - j : enter 
  - f : word forward 
  - b : C-left 
  - a : goto first char 
  - e : end 
  - w : delete word backword
